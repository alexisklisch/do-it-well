# Do it Well ✅

Another Todo List. **But** this have a motivational phrase.

## Instalation 💾

1. Clone the repository

    ```bash
    git clone 'http://github.com/alexisklisch/do-it-well'
    ```

2. Go to the directory and install the dependencies

    ```bash
    cd do-it-well &&
    yarn
    ```

## Preview the website 🔍

```bash
yarn run dev
```

## Build 🏗

```bash
yarn run build
```
